## Objective
To classify iris flowers into Setosa, Versicolor, and Virginica using petal and sepal measurements.

## Dataset
Iris dataset from scikit-learn (originally from UCI Machine Learning Repository).

## Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)

## Steps Performed
- Loaded and explored the dataset
- Visualized feature distributions
- Split data into training and test sets
- Trained classification models
- Evaluated accuracy
- Predicted a custom flower sample

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- scikit-learn

## Result
Achieved high accuracy in predicting iris flower species.
